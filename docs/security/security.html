externalusers
e.g : users_list = [r’Red\RedUser2’, r’Red\RedUser12’, r’mirje-pc\A’,
r’mirje-pcB’, r’John’, r’Prasad’, r’Mahesh’]

where: RedUser2, RedUser12 are belongs to AD ‘Red’ A, B are belongs to AD ‘mirje-pc’ John, Prasad, Mahesh are local users

local_usergroups (list) – complete list of local user groups
(Not required when updating external usergroup properties)

e.g : local_usergroups=[‘usergroup1’, ‘usegrouop2’]
